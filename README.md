#  Travel Website (Flask Project)

A dynamic travel website built using **Python Flask, HTML, and CSS** that allows users to explore travel destinations and get detailed information in a clean and user-friendly interface.

---

##  Features

*  Explore different travel destinations
*  Detailed information about places
*  Responsive and attractive UI using HTML & CSS
*  Backend powered by Flask (Python)

---

##  Tech Stack

* **Frontend:** HTML, CSS
* **Backend:** Python (Flask)

---

##  Project Structure

```
travel-website/
│── static/        # CSS, Images
│── templates/     # HTML files
│── app.py         # Main Flask app
│── requirements.txt
```

---

##  How to Run the Project

1. Clone the repository
2. Navigate to project folder
3. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
4. Run the app:

   ```
   python app.py
   ```
5. Open browser and go to:

   ```
   http://127.0.0.1:5000/
   ```
